Adds a string buffer to a response array.

Add buffer content to response array.

Parameters
bufferString to be added to the response array
responsePointer to array of strings (will be reallocated)
indexPointer to the current index in the response array

This function reallocates the response array to accommodate one more string, duplicates the provided buffer string at the current index position, adds a NULL terminator at the end of the array, and increments the index.

Returns
SUCCESS (0) if the buffer was added successfully, ERROR otherwise
